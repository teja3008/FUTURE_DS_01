# FUTURE_DS_01

This project is an **interactive Sales Analysis Dashboard** created using **Microsoft Excel** as part of **Task 1** of my internship at **Future Interns**.

## 🎯 Objective
To analyze sales data and generate meaningful business insights through an interactive dashboard.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Formulas
- KPI Cards

## 📂 Dataset Columns
- InvoiceNo
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

## 🧹 Data Cleaning
- Removed blank rows
- Removed missing descriptions
- Removed cancelled invoices
- Filtered invalid Quantity and UnitPrice values

## ➕ Helper Columns Created
- Sales
- Month
- Month_Year
- Year
- Day
- Hour

## 📈 Dashboard Includes

### KPI Cards
- Total Sales
- Total Orders
- Total Customers
- Average Order Value

### Charts
- Monthly Sales Trend
- Top 10 Products by Sales
- Sales by Country
- Orders by Day
- Sales by Hour

### Filters
- Country
- Year
- Month
